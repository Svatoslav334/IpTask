# IP Countries Task (C#)

Консольное приложение на C#, выполняющее анализ списка IP-адресов.

## Что делает программа

1. Загружает IP-адреса из файла `ips.txt`
2. Для каждого IP делает GET-запрос к API: https://ipinfo.io/{ip}/json
3. Определяет, к каким странам относятся IP и считает их количество
4. Находит страну с максимальным количеством IP и выводит города

## Запуск

```bash
dotnet restore
dotnet run
```
